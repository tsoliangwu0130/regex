Regex Note
==========

### Identifiers

| symbol | meaning |
|--------|---------|
| \  | escape character |
| \d | any number |
| \D | anything but a number |
| \s | space |
| \S | anything but a space |
| \w | any letter |
| \W | anything but a letter |
| \b | space around whole words |
| .  | any character, except for a new line |
| \. | period |
